# 01 基础与牌号认知

作为不锈钢外贸人，你不需要像材料学博士那样懂元素周期表，但你必须清楚每种材料的**核心卖点**和它们在**价格阶梯中的位置**。

## 1. 原理速成：不锈钢为什么不生锈？

- **碳钢（Carbon Steel）**：铁+碳，成本低，强度高，但极易生锈。
- **不锈钢（Stainless Steel）**：在铁中加入了**至少 10.5% 的铬 (Cr)**。铬会在钢表面形成一层致密的“氧化铬钝化膜”。这层膜如果被刮擦，在有氧气的情况下能自我修复。
- **为什么不锈钢也会生锈？** 
  - 遇到**氯离子（Cl-）**（如海水、盐雾、清洗剂），钝化膜会被破坏。
  - 表面附着了铁粉（如和碳钢一起用砂轮机切割）。

## 2. 常见系列与“赚钱”牌号

外贸中最常碰到的四大系列，它们的价格和性能阶梯如下：

### 📈 300 系列：外贸绝对主力（奥氏体）
*富含镍（Ni），无磁性，综合性能极佳。价格受 LME 镍价影响极大。*
- **304 (SUS304 / 1.4301)**：万金油，食品级首选。厨杂、水箱、建筑设备。
- **316L (SUS316L / 1.4404)**：相比 304 多了 **钼 (Mo)** 元素，抗海洋气候和化工腐蚀。
  * **实战案例**：客户做海边栏杆或游艇配件，首推 316L。如果是普通室内栏杆，推 304 即可。

### 📉 200 系列：低成本平替（奥氏体）
*以“锰 (Mn)”代替昂贵的“镍 (Ni)”，由于含镍极少，成本低，但容易生锈。*
- **201 / 202**：常用于室内装饰、低端五金、印度/中东/非洲的廉价厨具。
  * **实战案例**：客户询盘写要 304 钢管，但目标价低得离谱（比如 $1500/吨），你可以抛出试探：“Sir, your target price is workable for 201 J4/J1. For indoor decoration, 201 is completely enough. Should I quote 201 for your reference?”

### 🧲 400 系列：通常带磁性（铁素体/马氏体）
*不含镍或极少镍，成本比 304 低，导热性好，热膨胀率低。*
- **430 (1.4016)**：最常见的铁素体。用于家电面板（微波炉、冰箱外壳）、汽车排气管。
- **410 / 420**：含碳量高，常用来做需要硬度的刀具（马氏体）。

### 🛡️ 双相钢 (Duplex)：高端项目玩家
*奥氏体与铁素体各占一半，强度高，耐强腐蚀。*
- **2205 (1.4462) / 2507**：通常用于造纸、脱硫脱硝设备、海上油井。利润丰厚，多为工程单。

## 3. 标准与国际代号对照（防坑必背）

客户发来的询盘经常不按常理出牌，记住这几组等式，秒懂客户需求：

- **美标 vs 日标 vs 欧标**
  - `304` (美标/ASTM) = `SUS304` (日标/JIS) = `1.4301`或`X5CrNi18-10` (欧标/EN)
  - `316L` = `SUS316L` = `1.4404`或`X2CrNiMo17-12-2`
  - `430` = `SUS430` = `1.4016`

*小贴士：当欧洲客户询价 `1.4301` 时，务必确认材质单（MTC）是否需要满足 EN10088 标准。欧标对公差和力学性能的要求往往比美标严格。*
