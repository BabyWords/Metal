# 02 牌号与系列

## 1. 常见系列总览

- 200 系列: 以锰代镍，成本低，耐蚀性弱于 300 系。
- 300 系列: 奥氏体主力，综合性能好，外贸最常见。
- 400 系列: 多为铁素体或马氏体，通常有磁性。
- 双相钢: 高强高耐蚀，适合严苛工业环境。

## 2. 重点牌号

- 201: 低成本装饰与一般五金。
- 304/304L: 通用型首选。
- 316/316L: 海边、化工、医疗常用。
- 430: 家电和装饰常见。
- 2205: 双相钢代表型号。

## 3. 标准对照（高频）

- 304 = 1.4301 = SUS304
- 316L = 1.4404 = SUS316L
- 430 = 1.4016
- 2205 = 1.4462

## 4. 询盘中怎么看代号

示例: `SAC 316L 1.4404`

- `316L`: 美标名
- `1.4404`: 欧标材料号
- `SAC`: 常见为固溶退火状态（Solution Annealed Condition）
