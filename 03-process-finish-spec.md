# 03 工艺与表面

## 1. 热轧与冷轧

- 热轧（No.1）: 厚度大、成本低、表面粗。
- 冷轧（2B/BA）: 厚度更精准、表面更细。

## 2. 表面术语速查

- No.1: 热轧酸洗面。
- 2B: 冷轧常规面，应用最广。
- BA: 光亮退火面，亮度高。
- HL: 拉丝面，常用于电梯和装饰。
- 8K: 镜面抛光，高装饰需求。

## 3. BA 与镜面的区别

- BA 是工艺退火后形成的亮面。
- 镜面一般在 BA 或 2B 基材上再抛光。
- 镜面运输必须加厚保护膜，避免索赔。

## 4. 规格阅读示例

示例: `316L 1500mm BA`

- `316L`: 材质
- `1500mm`: 宽度（5 尺）
- `BA`: 表面

常见宽度: 1000, 1219, 1250, 1500 mm。
