# 05 外贸报价与风控

## 1. 报价结构

总价通常由三部分构成:

- 基价（随镍铬行情变化）
- 规格加价（厚度、宽度、特殊尺寸）
- 表面加价（BA、HL、8K、抗指纹等）

## 2. 理算与过磅

- 理算: 按理论尺寸和密度计算。
- 过磅: 按实际称重结算。

务必在合同中写清结算方式，否则容易造成毛利偏差。

## 3. 合同关键条款

- 材质标准: ASTM/EN/JIS/GB
- 边部: Mill Edge 或 Slit Edge
- 表面验收标准与贴膜要求
- 包装: Seaworthy Export Packing
- 交期与价格有效期

## 4. 常见索赔场景

- 表面划伤与镜面缺陷
- 实际厚度争议
- 海运受潮锈斑
- 包装破损导致边角变形

## 5. 业务动作清单

- 询盘先问终端用途
- 打样先锁定表面和硬度
- 下单前确认标准与公差
- 出货前留存照片与检验报告
